# REST Countries API with color theme switcher

Aplicação desenvolvida em React que permite pesquisar países, filtrar por região e visualizar informações detalhadas de cada país, incluindo população, capital, moedas, idiomas e países vizinhos. Os dados são consumidos dinamicamente através da REST Countries API e a aplicação conta com tema claro/escuro e layout totalmente responsivo para mobile, tablet e desktop.

# Live Demo
[View Project Online](https://rest-countries-api-two-smoky.vercel.app/)

## 🛠 Tecnologias Utilizadas
- React
- TypeScript
- HTML5
- Tailwind CSS
- React Router
- Context API
- Custom Hooks
- REST Countries API
- Fetch API

## 🚧 Desafios e Aprendizados

Durante o desenvolvimento deste projeto enfrentei alguns desafios importantes:

- Consumir dados de uma API externa utilizando Fetch API.
- Implementar rotas dinâmicas com React Router para navegação entre países.
- Buscar e exibir automaticamente informações dos países vizinhos.
- Criar e reutilizar um custom hook para centralizar a lógica de busca de dados.
- Implementar alternância entre tema claro e escuro utilizando Context API.
- Trabalhar com tipagem de dados utilizando TypeScript.
- Ajustar layout responsivo para diferentes tamanhos de tela utilizando Tailwind CSS.
- Refatorar estilos repetidos para melhorar organização e reutilização do código.
